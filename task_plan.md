## 游戏设计

### 核心概念

> 哆啦A梦的任意门可以通往任何地方。在这个游戏里，每一扇门背后，都藏着我们的一段回忆。配对成功，就能解锁一段属于我们的故事。

### 玩法

1. **记忆配对** — 一个 4×4 / 4×6 的翻牌记忆游戏
2. **配对机制** — 每对卡片是一张照片 + 对应的记忆标题/描述
3. **故事解锁** — 每成功配对一对，卡片翻转后展示一段甜蜜回忆
4. **进度系统** — 全部配对完成后，触发庆祝动画 + 展示完整时间轴
5. **难度选择** — 简单(4×3=6对) / 普通(4×4=8对) / 困难(4×6=12对)

### 主题设计

- **哆啦A梦蓝白红配色** — 复用 love-project 的 CSS 变量体系
- **任意门元素** — 卡片背面是任意门图案
- **飘浮装饰** — 竹蜻蜓、铜锣烧、铃铛等
- **庆祝动画** — 通关时心形粒子/彩带效果

### 素材引用

| 素材 | 来源 |
|------|------|
| 哆啦A梦 PNG | `images/doraemon.png` |
| 照片素材 | `images/*.webp`, `custom-images/**/*` |
| CSS 配色体系 | `css/style.css` 的 :root 变量 |
| 故事文本 | `js/main.js` 的 timelineData |
